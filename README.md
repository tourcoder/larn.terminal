# Larn Terminal Profile

Minimal macOS Terminal.app profile. Designed for a clean dark appearance.

## Usage

1. Download `Larn.terminal`
2. Double-click to import into Terminal
3. (Optional) Set as default profile

## Font

Rrecommend to use **[GoogleSansCode Nerd Font](https://github.com/tourcoder/GSC-Nerd-Font)** for the intended visual appearance.
